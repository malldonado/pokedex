## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Node.js installed on your local machine.

### Installation

1. Clone the repo

   ```sh
   npm install
```sh```

2. Install NPM packages

```sh
   npm install
```

3. Start the application
```sh
npm start
```
